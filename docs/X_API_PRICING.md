## 実測ログでの日次集計

推定ではなく実行時ログベースで日次コストを確認する:

```bash
# 今日
python3 analytics/daily_cost_report.py

# 昨日
python3 analytics/daily_cost_report.py --yesterday

# JSON
python3 analytics/daily_cost_report.py --yesterday --json

# 昨日分をDiscordへ通知
python3 analytics/daily_cost_report.py --yesterday --notify-discord
```

API利用ログ: `analytics/x_api_usage.jsonl`
