# hokke-x-profile-check

## 概要

OpenClawのブラウザ機能（Chrome Relay経由）を使い、**APIコストゼロ**でXのほっけプロフィールを確認するスキル。

check_engagement.py（API取得）の補完として位置づけ。両者を組み合わせて週次レビューを実施する。

---

## 確認項目

- フォロワー数・フォロー数
- 直近の投稿一覧（いいね・RT数が視覚的に確認できる）
- プロフィール表示の確認（アカウント凍結・制限フラグなど）

---

## 実行手順

```
1. ブラウザで https://x.com/cat_hokke を開く
2. フォロワー数をスクリーンショットまたはテキスト抽出
3. 直近5件の投稿を確認（いいね・RT数）
4. Discordに報告
```

---

## 利用タイミング

- 週次レビュー時に `check_engagement.py` と組み合わせて使う
- API取得でfailed件数が多い場合の確認
- フォロワー急増・急減の調査

---

## Discord報告フォーマット

```
🐾 ほっけ プロフィール確認

【フォロワー数】
X人（前回比: ±Y）

【直近5件】
1. "投稿テキスト" → いいね:N / RT:M
2. ...

【異常】
なし / [あれば記載]
```
