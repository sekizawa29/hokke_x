# ホッケリプライスキル

ホッケとしてターゲットアカウントにリプライする。

## プロジェクト

`~/pjt/hokke_x/reply_system/`

## いつ使うか

- ターゲットアカウントを発見したい時
- リプライを実行したい時
- リプライ状況を確認したい時

---

## コマンド

```bash
cd ~/pjt/hokke_x

# ターゲット発見（キーワード検索）
python3 x_cli.py reply discover

# リプライ実行
python3 x_cli.py reply run

# ドライラン（投稿しない）
python3 x_cli.py reply run --dry-run

# 状態確認
python3 x_cli.py reply status

# ターゲット手動追加
python3 x_cli.py reply add-target --username xxx --user-id yyy --category 猫系
```

---

## 運用ルール

- **1日10件**のリプライ（段階的に増加）
- **1日1アカウント1リプ**。広く浅く。
- リプ間隔: 最低2-3分空ける
- NGキーワード（政治・宗教・炎上等）はスキップ
- **実行速度**: スクリプト内にスリープがあるため、実行が遅くても問題なし

## ターゲットカテゴリ

| カテゴリ | 狙い |
|---------|------|
| 猫系 | 猫好きからのフォロー |
| 脱力系 | ペルソナと合うコミュニティ |
| メンタル系 | ターゲット層 |
| テック系 | 飼い主ネタの文脈 |

## リプライのトーン

PERSONA.md準拠。短く、ホッケらしく。

- 1〜2文（最大80文字程度）
- 「すごい」「いいね」だけの薄いリプはしない
- 猫の視点から人間を観察するような一言
- 攻撃的にならない。でも媚びない。
