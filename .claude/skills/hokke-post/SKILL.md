# ホッケ投稿スキル

ホッケとして今すぐツイートを投稿する。

## 重要

**テキスト生成はClaude自身が行う。外部API・anthropicパッケージは不要。**

**このスキル使用時はユーザーへの確認なしで投稿完了まで一貫して自動実行する。**

---

## 手順

### 1. 状況確認

```bash
cat ~/pjt/hokke_x/hook_performance.json
cat ~/pjt/hokke_x/PERSONA.md
```

- 直近のDROPカテゴリを把握する
- DROPが続いているカテゴリは避ける

### 2. カテゴリ選択

| カテゴリ | hook_category引数 |
|---------|-----------------|
| 脱力系 | 脱力系 |
| 猫写真 | 猫写真 |
| 鋭い一言 | 鋭い一言 |
| 日常観察 | 日常観察 |
| 時事ネタ | 時事ネタ |
| たまに有益 | たまに有益 |

### 3. テキスト生成

PERSONA.mdのルールに従いClaude自身が生成する。

**チェックリスト:**
- [ ] 140字以内
- [ ] 一人称は「僕」か使わない（「俺」「私」NG）
- [ ] 「〜にゃ」NG
- [ ] 絵文字は0〜1個
- [ ] 「頑張ろう」「自分を信じて」等のNGワードなし
- [ ] 短文・体言止め多め

### 4. 投稿実行

```bash
cd ~/pjt/hokke_x
python3 x_cli.py post \
  --text "投稿テキスト" \
  --hook-category "カテゴリ名"
```
